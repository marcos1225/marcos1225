### 🚀 Sobre mí
Ingeniero en Software y Desarrollador Full Stack de Guanacaste, Costa Rica. Mi enfoque principal es el **diseño de arquitecturas limpias, escalables y de alto rendimiento**, conectando ecosistemas Frontend modernos con Backends robustos y seguros.

- **Frontend Ecosystem:** Dominio avanzado en desarrollo SPA y SSR utilizando **React**, **Vue 3 (Quasar + Composables)** y **Astro** para máxima optimización SEO/PageSpeed.
- **Backend & Cloud:** Especializado en el ecosistema **.NET (.NET Core, JWT, Identity)**, APIs REST y bases de datos relacionales, respaldado por flujos automatizados de **CI/CD en Azure DevOps** y contenerización con **Docker**.
- **Filosofía de Trabajo:** Arquitectura limpia, Developer Experience (DX), desacoplamiento de lógica (Headless), patrones SOLID y código altamente reutilizable.

---

### 🧱 Guía de arquitectura que aplico
- UI en `components/` (solo visual, comunicación por `props`/`emits`).
- Lógica y estado en `composables/` o `hooks/` según el ecosistema.
- Diálogos dedicados y/o `Base*` reutilizables.
- Servicios de datos y peticiones HTTP en `services/`.
- Vistas en `views/` o `pages/`, layouts globales en `layouts/`.
- Funciones cortas, tipado estricto con TypeScript y arquitectura limpia.
